JSON to XML Converter

Convert your JSON data to its equivalent XML format with attribute support

JSON to XML converter transforms your JSON data to XML format. JSON keys become XML elements and JSON values become element values.

Special Syntax Support:

  • Begin JSON key with "-" to indicate XML attributes
  • Use "#text" in JSON to specify element text content

Syntax Examples

JSON with attributes and text:

{
  "book": {
    "-id": "123",
    "-category": "fiction",
    "title": {
      "-lang": "en",
      "#text": "The Great Gatsby"
    }
  }
}

Converts to XML:


  The Great Gatsby
